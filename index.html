Tentang Saya

Halo! Saya Rima Dwi Puspitasari, mahasiswi Teknik Informatika Universitas Lampung. Saya memiliki ketertarikan di bidang pengembangan web dan desain antarmuka pengguna (UI/UX).

Saya senang mempelajari teknologi baru dan membuat tampilan website yang mudah digunakan serta menarik bagi pengguna. Tujuan saya adalah mengembangkan solusi digital yang bermanfaat dan praktis untuk banyak orang.
